# Projeto-FullStack-App-Delivery-Oficial

O projeto consistiu na criação de um site institucional para um restaurante italiano fictício, desenvolvido com foco na experiência do usuário e na apresentação visual dos pratos e serviços oferecidos.
Foram utilizados HTML5 e CSS3 para a construção do front-end, priorizando uma estrutura semântica, responsividade e design atrativo inspirado em elementos da cultura italiana. O layout foi planejado para oferecer uma navegação intuitiva, com seções dedicadas ao cardápio, reservas, contato e informações sobre o restaurante.

Embora o projeto tenha sido implementado principalmente com tecnologias de front-end, também foram incluídos conceitos básicos de back-end — como a simulação de formulários de contato e reservas — utilizando apenas recursos de HTML e CSS para estruturar a interface e representar o comportamento esperado do sistema.

O objetivo principal foi exercitar boas práticas de desenvolvimento web, organização de código e estilização moderna, reforçando conhecimentos sobre estruturação de páginas, hierarquia de conteúdo, uso de classes e seletores CSS, além de técnicas de design responsivo.
